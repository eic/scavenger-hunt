title: Setup
---
If you have not done so already, please follow the instructions [here](https://eic.github.io/tutorial-setting-up-environment/setup.html) well before the start of the scavenger hunt to ensure your system is ready.

**Please note that installation can take roughly an hour to install and prepare. Allow a little time for setup before getting started.**

Where possible, it's recommended to run this on a computing cluster (E.g. JLab farm or BNL SDCC) as the installation may take up a few GB of storage. Downloaded files will also occupy more space.

You may also wish to try working through some of the other beginner/intermediate tutorials before starting the scavenger hunt. The tutorials can be found [here](https://eic.github.io/documentation/tutorials.html).
