title: "Instructor Notes"
---

This lesson is a self-guided scavenger hunt: learners complete a series of ePIC software tasks, each
of which yields a clue used to construct the filename for the final challenge.

## Before the session

- Ask learners to complete the [Setup](../learners/setup.md) page in advance. Installing `eic-shell`
  and downloading the container can take a long time, so it should not be done live.
- Post the target container version in the header of the
  [Scavenger Hunt Mattermost channel](https://chat.epic-eic.org/main/channels/scavenger-hunt), since
  Task 1 depends on it.

## Running the hunt

- Each task page embeds an answer-checker form that returns the clue for that task. Remind learners to
  record every clue, as all of them are needed for the final challenge.
- The tasks build on the EIC file-access, simulation, geometry, and analysis tutorials; keep those
  links handy for learners who need a refresher.
